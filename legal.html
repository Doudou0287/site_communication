ARTICLE 5 - CONTEXTE DU PROJET

Ce jeu est un projet étudiant du groupe 5B de l’UFR de Mathématique-Informatique de l’Université de Strasbourg. Il n’a pas pour but d’être commercialisé et est réalisé en accord avec le sujet suivant: L’objectif de ce projet est de revisiter un jeu de plateau de manière à le rendre collaboratif / multijoueurs en réseau. Le jeu sera dans un premier temps développé sur des clients lourds interagissant avec un serveur central. Dans un second temps, il sera adapté pour être jouable sur matériel mobile sans fil (tablette, tablette+raspberry) et/ou via une page web. Le jeu pourra utiliser des contrôles gestuels (leapmotion). L’authentification, la sauvegarde des scores, etc. seront réalisés via une base de données sur le serveur central.
